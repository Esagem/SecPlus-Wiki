# Transcripts

Raw auto-caption dumps from the Professor Messer SY0-701 YouTube playlist. One `.md` per exam objective, generated by the `transcripts.py` script in the project root.

**These files are immutable.** Treat them as raw source material. The LLM reads them to build distilled pages in `concepts/` and `objectives/`, but never edits a transcript file.

Inside each section file, every `## ` header is a single video. The transcript text follows as one paragraph (Professor Messer's auto-captions, lightly cleaned).

## Why they're here

The wiki exists so this raw firehose gets distilled into something studyable. A 5-minute Messer video on SQL Injection becomes a tight concept page, cross-referenced to its objective and quiz bank. The transcript stays in the wiki as the ground-truth source you can always trace back to.
